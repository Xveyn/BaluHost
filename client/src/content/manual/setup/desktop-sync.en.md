# Set Up Desktop Sync

This guide explains how to install the BaluDesk client and connect it to your BaluHost for automatic file synchronization.

## Prerequisites

- Admin or user access to BaluHost
- BaluDesk installed on your PC (download at GitHub: Xveyn/BaluDesk)
- BaluHost reachable on the local network or via VPN

## Steps

1. Open **BaluDesk** on your computer
2. Click **Add Server** and enter the server URL (e.g. `http://192.168.1.100:3001`)
3. Log in with your BaluHost credentials
4. Select the folders on the server you want to synchronize
5. Choose the local destination folder on your PC
6. BaluDesk starts the initial synchronization automatically
7. Sync status is shown in the system tray
