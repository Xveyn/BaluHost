# Connect Mobile App

This guide shows how to pair the BaluApp (Android) with your BaluHost using a QR code.

## Prerequisites

- Admin or user access to BaluHost
- BaluApp installed from the Google Play Store
- BaluHost running and reachable on the network

## Steps

1. Navigate to **Devices → Mobile Devices**
2. Click **Add Device**
3. Enter a name for the device (e.g. "My Smartphone")
4. A QR code is displayed — scan it with the BaluApp
5. The app connects automatically and prompts for a password to confirm pairing
6. After successful pairing the device appears in the device list
7. Optional: VPN configuration is automatically embedded for secure access when away from home
